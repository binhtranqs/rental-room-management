# Rental Room Frontend

React + Vite + TypeScript frontend for the rental room management app.

## Commands

Install dependencies:

```bash
npm install
```

Run local dev server:

```bash
npm run dev
```

Build production assets:

```bash
npm run build
```

Run lint:

```bash
npm run lint
```

## Environment

The frontend calls the backend at `http://localhost:8080` by default.

Override the API base URL with:

```bash
VITE_API_BASE_URL=http://localhost:8080 npm run dev
```
